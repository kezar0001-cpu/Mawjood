# Mawjood UX Principles

## 1. Arabic-first Design
- RTL everywhere  
- Right-aligned text  
- Search, filters, and key actions optimized for Arabic layouts  

## 2. Simplicity at All Costs
- No clutter  
- Minimal text  
- Clear and clean visuals  
- Easy scanning of categories and listings  

## 3. Efficiency
- One-tap actions (Call, WhatsApp, Maps)  
- High-speed search  
- Lightweight UI  

## 4. Familiarity
- Material icons  
- Recognizable patterns  
- Intuitive navigation  

## 5. Accessibility
- Font size minimum: 14px  
- High contrast text  
- Clear hit areas for all taps
